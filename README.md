# FitQuest

FitQuest คือระบบวางแผนและติดตามการออกกำลังกาย

ผู้ใช้สามารถสร้างโปรไฟล์ คำนวณ BMI เลือกโปรแกรมออกกำลังกาย สะสม EXP เพิ่ม Level และดูประวัติการออกกำลังกายได้

## Technologies

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Google Apps Script
- Google Sheets

## วิธีเปิดโปรเจกต์

### 1. เปิด Backend

เปิด Terminal แล้วเข้าโฟลเดอร์ `server`

```bash
cd server
npm install
node server.js
```

เมื่อเปิดสำเร็จจะเห็นข้อความ:

```text
Server running on http://localhost:3000
```

### 2. เปิด Frontend

เปิดไฟล์:

```text
code/index.html
```

ด้วย Live Server ใน Visual Studio Code
